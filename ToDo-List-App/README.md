# To-Do List Application

This project was created during my CodSoft Python Programming Internship. It is a simple desktop application that helps users keep track of their daily tasks. Users can add, update, delete, and mark tasks as completed through an easy-to-use interface.

## Features
- Add new tasks
- View the task list
- Update existing tasks
- Delete tasks
- Mark tasks as completed
- Save tasks automatically

## Technologies Used
- Python
- Tkinter
- JSON

## Why I Built This Project
I developed this project to improve my understanding of Python programming and learn how to create desktop applications using Tkinter. It also helped me practice file handling and organizing data in a simple way.
